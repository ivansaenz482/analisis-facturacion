
Análisis de facturación de negocio simulado

 Objetivo
Demostrar habilidades en **análisis de datos aplicado a ventas y facturación**, combinando mi experiencia comercial con herramientas tecnológicas.

 Resultados clave
- Facturación total: **$9,845**
- Producto estrella: **Laptop** (5 unidades vendidas)
- Cliente top: **Cliente A** ($2,690 en compras)
- Zona con mayor facturación: **Norte** (38.5% del total)

 Tecnologías usadas
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook / Google Colab
- CSV para almacenamiento de datos

 Archivos incluidos
| Archivo | Descripción |
|---------|-------------|
| `ventas_simuladas.csv` | Datos de 6 meses (productos, clientes, zonas) |
| `analisis_ventas.ipynb` | Código completo del análisis |
| `dashboard_facturacion.png` | Visualización de resultados |

 Recomendaciones para el negocio
1. Fidelizar al **Cliente A** con descuentos exclusivos
2. Reforzar ventas en la **zona Centro** (menor facturación)
3. Crear promociones en **enero** (mes de menor venta)

 Cómo ejecutar este proyecto
1. Abrir [Google Colab](https://colab.research.google.com)
2. Subir el archivo `ventas_simuladas.csv`
3. Copiar y pegar el código Python
4. Ejecutar celda por celda
